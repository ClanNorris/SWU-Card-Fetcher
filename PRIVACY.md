# Privacy Policy — SWU Card Fetcher

*Last updated: June 22, 2026*

SWU Card Fetcher ("the Bot") is a Discord bot that retrieves Star Wars Unlimited trading card information from the public swu-db.com API and displays results in Discord chat.

## Data We Collect

The Bot does not collect, store, or log any personal information about Discord users.

When the Bot detects a card lookup request (via `[[Card Name]]` syntax or the `/swucard` slash command), it reads only the card name from your message. This card name is passed to the swu-db.com public API to retrieve card data. No message content, user IDs, usernames, or server information is retained after the request is complete.

## Data We Do Not Collect

- Message content beyond the card name in a lookup request
- User IDs, usernames, or profile information
- Server or channel information
- Usage history or analytics

## Third-Party Services

Card data is retrieved from [swu-db.com](https://swu-db.com). Please refer to their privacy policy for information about how that service handles requests.

## Server Configuration

Server administrators may configure bot settings via slash commands. These settings (such as default search preferences) are stored per-server and contain no personal user data.
